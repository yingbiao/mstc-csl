# 墨尔本神学院中文部Zotero样式

# 安装

1. 下载文件 [melbourne-school-of-theology-chinese.csl](https://github.com/yingbiao/mstc-csl/raw/master/melbourne-school-of-theology-chinese.csl), 请确保文件的后缀是.csl
2. 打开Zotero的Preferences
3. 在Cite标签页添加下载的样式

# 注意事项
## 语言设置
简体中文的参考资料请把语言（Language）一项设置成`zh-CN`，  
繁体中文的参考资料请把语言（Language）一项设置成`zh-TW`，  
语言未注明，不是`zh-CN`，`zh-TW`，则系统自动使用`en-US`。

## 中文名字
中文的名字请用一个输入框，姓和名输入在一起，不要拆开。

# 注脚和参考书目格式
## 基本格式   
**注脚格式**   
>Firstname Lastname, *Title* (City: Publisher, Year), pages cited.   
>姓名：《書名》（出版城市：出版社，年份），頁數。

**重复引用格式**    
>Lastname, *Short Title*, pages cited.    
>姓名：《書名》，頁數。

**参考书目格式**    
>Lastname, Firstname. *Title* (City: Publisher, Year).    
>姓名：《書名》（出版城市：出版社，年份）。

## 当有更多信息时
**注脚格式**    
>Firstname Lastname, *Title* (ed. Firstname Lastname; trans. Firstname Lastname; number of volumes; edition; Series Name  series number; City: Publisher, Year), URL (accessed Access Date), volume:pages cited.     
>姓名：《書名》（姓名編，姓名譯，總冊數，版數，叢書名 數目，出版城市：出版社，年份），網址（參閱日期 參閱），冊數：頁數。  

**重复引用格式**    
>Lastname, *Short Title*, volume: pages cited.    
>姓名：《書名》，冊數：頁數。    

**参考书目格式**    
>Lastname, Firstname. *Title* (edited by Firstname Lastname; translated by Firstname Lastname; number of volumes; edition; Series Name series number; City: Publisher, Year), URL (accessed Access Date).     
>姓名：《書名》（姓名編，姓名譯，總冊數，版數，叢書名 數目，出版城市：出版社，年份），網址（參閱日期 參閱）。
