# 墨尔本神学院中文部Zotero样式

## 安装

1. 下载文件 [melbourne-school-of-theology-chinese.csl](https://github.com/yingbiao/mstc-csl/raw/master/melbourne-school-of-theology-chinese.csl)
2. 打开Zotero的Preferences
3. 在Cite标签页添加下载的样式
